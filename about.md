# AutoLeaderboard

This is a fork of [M336G/AutoLeaderboard](https://github.com/M336G/AutoLeaderboard).
Automatically update the level leaderboards so that you don't have to do it yourself.

## Notice

I prefer to use this fork over the original or the BetterInfo version.
