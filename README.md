# AutoLeaderboard

This is a fork of [M336G/AutoLeaderboard](https://github.com/M336G/AutoLeaderboard).
Automatically update the level leaderboards so that you don't have to do it yourself.

## Notice

I prefer to use this fork over the original or the BetterInfo version.

## Deprecation

> [!WARNING]
> Due to some issues regarding the mod, I have decided to **deprecate** it. If you still wish to use this mod's feature, enable **[BetterInfo](https://geode-sdk.org/mods/cvolton.betterinfo)'s "Auto submit to leaderboards"** feature.
> ![Deprecation banner](https://github.com/user-attachments/assets/84880219-5810-4763-a3f7-8ad0389f5920)
